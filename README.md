# ♟️ Python Chess Game

A two-player chess game built in Python. The project focuses on implementing core chess logic, turn-based gameplay, and a functional board system. Players interact with the game through simple inputs, and the visual version uses **qtido** to render the chessboard and pieces.

---

## 🎮 Features

- Two-player turn-based gameplay  
- Chessboard representation and dynamic updates  
- Piece movement system  
- Input-based move handling  
- Graphical version using **qtido** for rendering  
- Modular structure, making it easy to extend  

---

## 🖱️ How to play

Players take turns selecting a piece and choosing its destination square. After each move, the board is updated visually. The game continues until players decide to stop.
